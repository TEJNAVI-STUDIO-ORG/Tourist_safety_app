
# Contributing to TouristSafe

Thank you for helping improve TouristSafe! This project is open to contributions from developers, designers, testers, and writers.

## How to Contribute

1. Fork the repository.
2. Create a new branch:
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. Make your changes.
4. Run tests and verify functionality.
5. Submit a pull request with a clear description.

## Contribution Areas

- UI/UX improvements
- Firebase integration and backend logic
- Location and geofence behavior
- SOS and SMS handling
- Documentation updates
- Tests and bug fixes

## Reporting Issues

Open issues for:
- bugs
- missing features
- unclear setup steps
- documentation problems

## Code Style

- Follow Flutter and Dart best practices.
- Use clear variable names.
- Keep widgets small and reusable.
- Add comments for complex logic.
